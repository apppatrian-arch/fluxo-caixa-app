# Fluxo de Caixa Pessoal

Aplicativo web mobile-first para controle de recebimentos e pagamentos.

## Funcionalidades

- Login por usuário e senha
- Lançamentos de recebimentos e pagamentos
- Cadastro de pessoas (nome e telefone)
- Categorias personalizáveis
- Saldo em caixa em tempo real
- Gráficos de pizza e barras por mês
- Histórico mensal
- Dados sincronizados via GitHub API
- Funciona offline (PWA)

## Tecnologias

- HTML / CSS / JavaScript puro
- Chart.js
- GitHub API (armazenamento de dados)
- Service Worker (PWA)
